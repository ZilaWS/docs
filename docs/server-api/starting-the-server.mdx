# Starting the server

## Secure

```ts
const server = new ZilaServer({
  port: 6594,
  logger: true, //Enables logging
  verbose: true, //Enables verbose logging
  https: {
    pathToCert: join(__dirname, "cert/cert.pem"), //Path to the SSL cert file
    pathToKey: join(__dirname, "cert/key.pem"), //Path to the SSL key file
    passphrase: "asdASD123", //Passphrase for the certificate
  },
});
```

## Non-secure

```ts
const server = new ZilaServer({
  port: 6589,
  logger: true, //Enables logging
  verbose: true //Enables verbose logging
});
```
